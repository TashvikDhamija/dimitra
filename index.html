Abstract

Dimitra is an audio-driven diffusion model designed to generate expressive talking heads that synchronize lip movements, facial expressions, and emotions with the input audio. We demonstrate how Dimitra produces more natural and realistic talking head videos compared to existing models. This page provides supplementary materials, including example videos, model architecture, and ablation studies.
